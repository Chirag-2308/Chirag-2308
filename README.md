### 🏭 What I'm Currently Working On

**Predictive Maintenance System — Industrial AI**

Industrial machines fail silently — until they don't, and everything grinds to a halt. I'm building an ML-powered predictive maintenance system that catches those failures early using sensor data, anomaly detection, and smart data pipelines.

The mission: shift maintenance from reactive to predictive. Less downtime. Less guessing. More data-driven decisions. Think of it as teaching machines to know when they're about to have a bad day — before it happens. 🤖

Same curiosity. Same mindset. Different stakes — and honestly, that's what makes it exciting.
